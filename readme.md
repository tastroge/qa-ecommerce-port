# Portfólio de QA Manual – E-commerce

Projeto prático de QA Manual focado na validação do fluxo de compra em um e-commerce, desde a seleção de produtos até a finalização do pagamento.

## Escopo testado
- Carrinho de compras
- Checkout
- Pagamento
- Experiência do usuário

## Objetivo
Garantir a qualidade funcional e a boa experiência do usuário em um ambiente de e-commerce.
